CTXMAMLogger

Superclass:
NSObject
Declared In:
CTXMAMLogger.h

Introduction

This is a wrapper for the CTXLOG calls and is used to provide interfaces used for logging messages into the Citrix CTXMAM Framework logging facility.

CTXMAMLog_CriticalErrorFrom:andFile:andFunction:atLine:withFormat:

+(void) CTXMAMLog_CriticalErrorFrom: (NSString *)source 
        andFile:(nullable NSString *)fileName andFunction:(nullable NSString *)functionName 
        atLine:(int) lineNumber withFormat:(NSString *)fmt, ...;
Parameters
source

A string object which describes the module name of the log message.

fileName

A string object which contains the name of the source file where the message resides - use __FILE__ macro to get the current file name.

functionName

A string object which contains the name of the function where the message comes from - use __FUNCTION__ macro to get the current function name.

lineNumber

An int which represents the line number of the source file where the message resides - use __LINE__ macro to get the current line number.

fmt

A standard format string which contains the customized message to be logged out to the logger.

Discussion

CTXMAMLog_CriticalErrorFrom function logs the specified message as a Critical Error.
